

# Provo Digital Wellness

## Project Overview
Provo Digital Wellness is a community-focused web project designed for the residents of Provo, Utah. 
Its primary goal is to raise awareness about the impact of social media on mental health. By leveraging data and providing resources, 
the website aims to foster healthier social media habits and offer insights into digital wellness.

## Key Features
- **Community-Oriented Design:** Tailored for Provo, Utah residents, focusing on local needs and interests.


- **Interactive Survey:** Users participate in a survey, contributing data on social media usage and its effects.


- **Dynamic Dashboard:** Visualizes survey data to depict the positive and negative impacts of social media. 
It includes a public Tableau dashboard with filters for in-depth analysis.


- **User Accounts:** Features a login system allowing account creation, password editing, and profile management. 
A superuser role is implemented for comprehensive site management, including viewing all participant data.


- **Resource Hub:** The homepage offers various resources and links to promote better digital wellness practices.


- **Data Management:** Automatically updates the database with new survey responses, maintaining a current dataset for analysis.

## Installation
This project requires Node.js and additonally to access our database Pgadmin PostgreSQL is required.

1. Clone the repository:
   ```
   git clone [repository URL]
   ```
2. Install dependencies:
   ```
   npm install
   ```

## Usage
To run the project:
1. Start the server:
